# 有品商城 - 企业级商城后台管理系统
## 📝 项目前言
感谢 **WangJian3306** 提供的[后端API](https://github.com/WangJian3306/vue3_admin_backend)

**有品商城**是一款基于最新前端技术栈构建的现代化、高性能商城后台管理解决方案。
本项目深度还原了企业级中后台的开发流程与业务逻辑。在构建过程中，我不仅追求界面的美观与交互的流畅，更在工程化标准、权限控制安全性以及复杂数据可视化方面进行了深度实践。

## 🛠 核心技术栈
本项目紧跟前端社区潮流，采用了全家桶式的现代化开发组合：
  - 框架核心：Vue 3 + TypeScript（强类型校验，提升代码健壮性）
  - 构建工具：Vite（极速的热更新与打包体验）
  - 状态管理：Pinia（轻量、模块化的新一代状态管理）
  - 路由方案：Vue-Router 4
  - UI 组件库：Element-Plus（深度定制主题与按需引入）
  - 网络交互：Axios（二次封装实现统一拦截与错误处理）

## ✨ 项目亮点
为了应对真实业务挑战，本项目实现了以下核心特色功能：

🔐 **严密的权限控制体系**
  - 实现了完善的 RBAC (Role-Based Access Control) 权限模型：
  - 菜单权限：根据用户角色动态生成路由表，确保“无权不可见”。
  - 按钮权限：通过自定义指令，实现细粒度到每一个操作按钮的控制，确保“无权不可动”。

📊 **炫酷的数据大屏**
  - 针对电商经营数据，打造了极具视觉冲击力的自适应数据可视化大屏：
  - 运用 ECharts 进行多维度数据展示。
  - 攻克了多种比例下的 等比例缩放适配方案，确保在不同分辨率显示器下均能完美呈现。

🎨 **极致的个性化体验**
  - 一键换肤：支持全局主题色配置，满足不同企业的品牌定制需求。
  - 暗黑模式：内置暗黑模式切换逻辑，适配多种办公场景，提升用户交互体验。

🚀 **工程化实践**
  - Axios 二次封装：统一处理请求/响应拦截、Loading 状态同步及全局错误反馈。
